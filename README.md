Why so sad
=========

A simple website to brighten up your day when you're down. [Visit Website](http://whysosad.win)

## Installation

- Clone `git clone git@github.com:siebevd/whysosad.git`
- Install packages : `npm install`
- Copy `config.example.js` > `config.js` and fill in your credentials
- Run dev build ( & server ): `npm run dev`
- Build for production: `npm run build`

## Tools used
- [Rollup](https://rollupjs.org)
- [Preact](https://preactjs.com) (Since this is such a small website, let's keep it lightweight and use preact instead of react)
- [Postcss](http://postcss.org/)
